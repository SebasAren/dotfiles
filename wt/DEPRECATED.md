# Deprecated: Worktrunk (wt) Package

This stow package is deprecated. The project has migrated from git worktrees (Worktrunk/wt) to jj (jujutsu) as version control porcelain.

- The `wt` CLI is no longer used
- Git worktrees are no longer used for parallel work
- Commit messages are now generated via jj + Pi LLM
- The commit skill and tdd-plan skill have been updated for jj

This directory is kept for reference only. Do not `stow wt`.
